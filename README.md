
```js
// available mainnet networks are:
const myNetworks = [
	Network.OPT_MAINNET,
	Network.ARB_MAINNET,
	Network.ETH_MAINNET,
	Network.MATIC_MAINNET,

```

- To run the program:

```bash
node main.js
```
